Política de Privacidade

Última atualização: 30 de dezembro de 2024

1. Introdução

O CS Implantação ("Sistema" ou "Aplicação") é uma plataforma interna da Pacto Soluções destinada ao gerenciamento de implantações de clientes. Esta Política de Privacidade descreve de forma clara e transparente como coletamos, usamos, armazenamos e protegemos as informações dos usuários, especialmente dados obtidos através do Google OAuth 2.0.

2. Dados Acessados do Google (Google User Data)

Nossa aplicação acessa os seguintes dados através da API do Google:

  • E-mail do usuário (scope: email) - Usado para autenticação e identificação única do usuário
  • Perfil básico (scope: profile) - Inclui nome completo e foto de perfil
  • Identificador único do Google (sub) - Usado para vincular a conta Google ao perfil interno

Importante: Nosso acesso é restrito exclusivamente a contas corporativas @pactosolucoes.com.br. Não acessamos dados de contas pessoais do Google.

3. Como Usamos os Dados do Google

Os dados obtidos do Google são utilizados exclusivamente para:

  • Autenticação segura - Verificar a identidade do usuário sem necessidade de senha adicional
  • Identificação do usuário - Exibir nome e foto no sistema para facilitar a comunicação entre equipes
  • Controle de acesso - Garantir que apenas funcionários autorizados da Pacto Soluções acessem o sistema
  • Registro de atividades - Associar ações no sistema (comentários, atualizações de status) ao usuário responsável

Não utilizamos dados do Google para: Marketing, publicidade, treinamento de IA/ML, ou qualquer finalidade não relacionada à operação interna do sistema.

Declaração de Uso Limitado

O uso de informações recebidas das APIs do Google pelo CS Implantação está em conformidade com a Google API Services User Data Policy , incluindo os requisitos de Uso Limitado. Nosso uso de dados do Google é limitado exclusivamente às práticas divulgadas nesta política de privacidade.

4. Dados Adicionais Coletados pelo Sistema

Além dos dados do Google, coletamos e armazenamos:

  • Dados de implantações - Informações sobre clientes e projetos gerenciados
  • Comentários e notas - Registros de comunicação interna sobre implantações
  • Logs de atividade - Histórico de ações realizadas no sistema (auditoria)
  • Preferências do usuário - Configurações de interface e notificações

5. Compartilhamento de Dados

Não compartilhamos dados do Google ou dados do sistema com terceiros externos.

O acesso aos dados é restrito a:

  • Funcionários autorizados da Pacto Soluções com perfil de acesso apropriado
  • Administradores do sistema para manutenção e suporte técnico

Dados podem ser compartilhados apenas se exigido por lei ou ordem judicial.

6. Armazenamento e Proteção de Dados

Implementamos as seguintes medidas de segurança:

  • Criptografia em trânsito - Todas as conexões usam HTTPS/TLS
  • Criptografia em repouso - Dados sensíveis são criptografados no banco de dados
  • Autenticação OAuth 2.0 - Protocolo seguro do Google para login
  • Controle de acesso baseado em perfis - Usuários só acessam dados necessários para sua função
  • Logs de auditoria - Registro de todas as ações críticas no sistema
  • Backups regulares - Proteção contra perda de dados
  • Infraestrutura segura - Hospedagem em provedores certificados (Railway)

7. Retenção e Exclusão de Dados

Período de retenção:

  • Dados de usuários ativos são mantidos enquanto a conta estiver ativa
  • Dados de ex-funcionários são mantidos por até 5 anos para fins de auditoria e compliance
  • Logs de atividade são mantidos por 2 anos

Solicitação de exclusão:

Usuários podem solicitar a exclusão de seus dados pessoais a qualquer momento entrando em contato com suporte@pactosolucoes.com.br. Processaremos a solicitação em até 30 dias, exceto dados que devem ser mantidos por obrigação legal.

8. Direitos do Usuário (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD), você tem direito a:

  • Acesso - Solicitar cópia dos seus dados pessoais
  • Correção - Atualizar dados incorretos ou desatualizados
  • Exclusão - Solicitar remoção dos seus dados (sujeito a obrigações legais)
  • Portabilidade - Receber seus dados em formato estruturado
  • Revogação de consentimento - Revogar autorização de uso de dados

Para exercer esses direitos, entre em contato através do e-mail: suporte@pactosolucoes.com.br

9. Conformidade com Políticas do Google

Esta aplicação está em conformidade com:

  • Google API Services User Data Policy
  • Google APIs Terms of Service

Nosso uso de dados do Google é limitado, transparente e segue o princípio de "acesso mínimo necessário".

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos os usuários sobre mudanças significativas através do sistema. A data da última atualização está indicada no topo desta página.

11. Contato e Encarregado de Dados (DPO)

Para questões sobre privacidade e proteção de dados:

  • E-mail: suporte@pactosolucoes.com.br
