1. Introdução

O CS Implantação ("Sistema") é uma aplicação interna da Pacto Soluções destinada ao gerenciamento de implantações de clientes. Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários.

2. Dados Coletados

Coletamos os seguintes dados através do login com Google:

  • E-mail corporativo (@pactosolucoes.com.br)
  • Nome do usuário (conforme cadastro no Google)
  • Identificador único do Google (sub)

Também coletamos dados de uso do sistema, como ações realizadas em implantações, comentários e registros de atividade para fins de auditoria e histórico.

3. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticação e controle de acesso ao sistema
  • Identificação do usuário responsável por ações no sistema
  • Registro de histórico e auditoria de atividades
  • Comunicação interna relacionada às implantações

4. Compartilhamento de Dados

Os dados não são compartilhados com terceiros externos à Pacto Soluções. O acesso é restrito a funcionários autorizados da empresa.

5. Segurança

Implementamos medidas de segurança técnicas e organizacionais para proteger os dados, incluindo:

  • Conexões criptografadas (HTTPS)
  • Autenticação segura via Google OAuth 2.0
  • Controle de acesso baseado em perfis
  • Logs de auditoria

6. Retenção de Dados

Os dados são mantidos enquanto o usuário estiver ativo na empresa ou conforme necessário para fins de auditoria e compliance.

7. Direitos do Usuário

Os usuários podem solicitar informações sobre seus dados ou exclusão dos mesmos entrando em contato com o administrador do sistema.

8. Contato

Para dúvidas sobre esta política, entre em contato através do e-mail: suporte@pactosolucoes.com.br
